Ngx Image Zoom

This is used to apply zooming effect over an image.

It has four types of mode:- hover,click,toggle,hover-freeze

  1. Install- npm install ngx-image-zoom --save
  2. import NgxImageZoomModule in import array.

For further reference click here: NgxImageZoom
